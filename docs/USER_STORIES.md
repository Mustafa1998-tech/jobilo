## Persona 1: Freelancer (مستقل)

### US-FR-1: Registration & Onboarding

| ID | Story | Priority |
|----|-------|----------|
| US-FR-1.1 | As a **freelancer**, I want to **register with my email and password** so that **I can create my account on the platform** | P0 |
| US-FR-1.2 | As a **freelancer**, I want to **register using my Google account** so that **I don't have to remember another password** | P0 |
| US-FR-1.3 | As a **freelancer**, I want to **verify my email via OTP** so that **my account is secure and active** | P0 |
| US-FR-1.4 | As a **freelancer**, I want to **reset my password if I forget it** so that **I can regain access to my account** | P0 |

### US-FR-2: Profile Management

| ID | Story | Priority |
|----|-------|----------|
| US-FR-2.1 | As a **freelancer**, I want to **create a professional profile with my photo, bio, and skills** so that **clients can see my expertise** | P0 |
| US-FR-2.2 | As a **freelancer**, I want to **add my portfolio projects with images and descriptions** so that **I can showcase my past work** | P0 |
| US-FR-2.3 | As a **freelancer**, I want to **add my work experience, education, and certifications** so that **I build credibility with clients** | P1 |
| US-FR-2.4 | As a **freelancer**, I want to **set my hourly and fixed project rates** so that **clients know my pricing upfront** | P0 |
| US-FR-2.5 | As a **freelancer**, I want to **see my profile strength percentage** so that **I know what to improve** | P0 |
| US-FR-2.6 | As a **freelancer**, I want to **get AI suggestions to improve my profile** so that **I appear more attractive to clients** | P1 |

### US-FR-3: Browse & Discover Projects

| ID | Story | Priority |
|----|-------|----------|
| US-FR-3.1 | As a **freelancer**, I want to **browse all open projects with pagination** so that **I can find work opportunities** | P0 |
| US-FR-3.2 | As a **freelancer**, I want to **search projects by keywords** so that **I find relevant projects quickly** | P0 |
| US-FR-3.3 | As a **freelancer**, I want to **filter projects by category, budget, duration, and experience level** so that **I narrow down the best opportunities** | P0 |
| US-FR-3.4 | As a **freelancer**, I want to **save/bookmark interesting projects** so that **I can review and apply later** | P0 |
| US-FR-3.5 | As a **freelancer**, I want to **get AI-powered project recommendations** so that **I see projects matching my skills** | P1 |

### US-FR-4: Submit & Manage Proposals

| ID | Story | Priority |
|----|-------|----------|
| US-FR-4.1 | As a **freelancer**, I want to **submit a proposal with a cover letter and my bid amount** so that **I can apply for a project** | P0 |
| US-FR-4.2 | As a **freelancer**, I want to **attach files to my proposal** so that **I can share relevant samples** | P0 |
| US-FR-4.3 | As a **freelancer**, I want to **use AI to draft my proposal** so that **I save time and submit better proposals** | P1 |
| US-FR-4.4 | As a **freelancer**, I want to **view all my submitted proposals in one place** so that **I can track their status** | P0 |
| US-FR-4.5 | As a **freelancer**, I want to **withdraw my proposal** so that **I can cancel if I'm no longer interested** | P0 |
| US-FR-4.6 | As a **freelancer**, I want to **get notified when my proposal is accepted or rejected** so that **I know the outcome quickly** | P0 |

### US-FR-5: Communicate with Clients

| ID | Story | Priority |
|----|-------|----------|
| US-FR-5.1 | As a **freelancer**, I want to **chat with clients in real-time** so that **we can discuss project details** | P0 |
| US-FR-5.2 | As a **freelancer**, I want to **send files in chat** so that **I can share work samples and drafts** | P0 |
| US-FR-5.3 | As a **freelancer**, I want to **see when my messages are read** so that **I know the client has seen them** | P0 |

### US-FR-6: Receive Reviews

| ID | Story | Priority |
|----|-------|----------|
| US-FR-6.1 | As a **freelancer**, I want to **receive ratings and reviews from clients** so that **I build my reputation** | P0 |
| US-FR-6.2 | As a **freelancer**, I want to **see my average rating on my profile** so that **new clients trust me** | P0 |
| US-FR-6.3 | As a **freelancer**, I want to **reply to client reviews** so that **I can address feedback** | P1 |

### US-CL-1: Registration & Onboarding

| ID | Story | Priority |
|----|-------|----------|

### US-CL-2: Post & Manage Projects

| ID | Story | Priority |
|----|-------|----------|

| US-CL-2.3 | As a **client**, I want to **save my project as a draft** so that **I can publish it later** | P0 |
| US-CL-2.4 | As a **client**, I want to **preview my project before publishing** so that **I can review and correct any mistakes** | P0 |
| US-CL-2.5 | As a **client**, I want to **edit my project after posting** so that **I can update details** | P0 |
| US-CL-2.6 | As a **client**, I want to **close my project early** so that **I stop receiving proposals** | P0 |

| ID | Story | Priority |
|----|-------|----------|

| US-CL-3.2 | As a **client**, I want to **shortlist promising proposals** so that **I keep track of top candidates** | P0 |
| US-CL-3.3 | As a **client**, I want to **view the freelancer's profile and portfolio** so that **I assess their skills** | P0 |
| US-CL-3.4 | As a **client**, I want to **accept a proposal** so that **I start working with that freelancer** | P0 |
| US-CL-3.5 | As a **client**, I want to **reject proposals with feedback** so that **freelancers know why they weren't selected** | P0 |

| US-CL-4.2 | As a **client**, I want to **discuss project details in real-time** so that **we align on expectations** | P0 |

### US-CL-5: Leave Reviews

| ID | Story | Priority |
|----|-------|----------|

| US-CL-5.2 | As a **client**, I want to **rate specific aspects (quality, communication, timeliness)** so that **reviews are detailed** | P0 |
