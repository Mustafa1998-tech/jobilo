## تفاصيل كل صفحة

### 1. /admin/login — صفحة تسجيل دخول المشرفين

**المكونات:**
- شعار Jobilo مع نص "لوحة الإدارة"
- حقل البريد الإلكتروني
- حقل كلمة المرور
- خيار "تذكرني"
- زر دخول
- رسالة خطأ (بيانات غير صحيحة / حساب مقفل)
- رابط "نسيت كلمة المرور"
- Captcha (hCaptcha) بعد 3 محاولات فاشلة
- Modal إدخال رمز 2FA (في المرحلة الثانية)

**الحالات:**
| الحالة | الوصف |
|--------|-------|
| Default | النموذج فارغ وجاهز للإدخال |
| Loading | تعطيل الأزرار مع Spinner |
| Error | رسالة خطأ تحت الحقل المعني |
| Locked | رسالة "حسابك مقفل لمدة 15 دقيقة" |
| 2FA | طلب إدخال رمز المصادقة الثنائية |

### 3. /admin/users — قائمة المستخدمين

**المكونات:**
- شريط بحث (بحث بالاسم أو البريد أو ID)
- أزرار فلترة (الدور، الحالة، تاريخ التسجيل)
- ترتيب (بأي عمود)
- جدول المستخدمين:
  | المستخدم | البريد | الدور | الحالة | تاريخ التسجيل | آخر نشاط | الإجراءات |
  |----------|--------|-------|--------|---------------|----------|----------|
- أزرار إجراءات سريعة لكل مستخدم (تعديل، حظر، حذف)
- Pagination (10/20/50/100)
- إمكانية تحديد عدة مستخدمين (Batch Actions)
- تصدير القائمة (CSV/PDF)

**Batch Actions:**
- تفعيل المحددين
- تعطيل المحددين
- حظر المحددين
- حذف المحددين
- إرسال إشعار جماعي

**فلترة متقدمة:**
```

│ الدور: [▼ جميع الأدوار ]  الحالة: [▼ جميع الحالات ] │
```

### 8. /admin/settings — الإعدادات

صفحة إعدادات تحتوي على تبويبات (Tabs):

**General (إعدادات المنصة):**
- اسم المنصة (عربي/إنجليزي)
- وصف المنصة
- الشعار
- الفافيكون
- اللغة الافتراضية
- العملة الافتراضية
- نسبة عمولة المنصة
- حد المبلغ الأدنى للسحب
- وضع الصيانة (تشغيل/إيقاف)
- الحد الأقصى للمشاريع النشطة لكل عميل

**Email (البريد الإلكتروني):**
- SMTP Host / Port / Username / Password
- From Name / From Email
- تمكين/تعطيل إرسال الإيميلات
- قوالب الإيميلات (WYSIWYG Editor)

**Storage (التخزين):**
- نوع التخزين (محلي / S3 / Cloudinary)
- إعدادات Cloudinary (Cloud Name, API Key, API Secret)
- أقصى حجم للملف
- الصيغ المسموح بها
- ضغط الصور

**AI (الذكاء الاصطناعي):**
- مفتاح OpenAI API
- الموديل (GPT-4o-mini / GPT-4o)
- الحد الأقصى للـ Tokens
- درجة الحرارة (Temperature)
- تفعيل/تعطيل ميزات AI:
  - ☑ كاتب العروض
  - ☑ توصيات المستقلين
  - ☑ تحسين السيرة الذاتية
  - ☑ كشف الاحتيال
  - ☑ الدردشة الذكية

**Notifications (الإشعارات):**
- إعدادات الإيميل (تشغيل/إيقاف كل نوع)
- إعدادات Push Notification
- Firebase / Web Push Config

**SEO (تحسين محركات البحث):**
- عنوان الموقع
- الوصف
- الكلمات المفتاحية
- Google Analytics ID
- Google Search Console
- Robots.txt
- Sitemap

**Security (الأمان):**
- عدد محاولات تسجيل الدخول المسموح بها قبل القفل
- مدة القفل (بالدقائق)
- تعقيد كلمة المرور (أحرف كبيرة/صغيرة/أرقام/رموز)
- صلاحية التوكن (Access: دقائق، Refresh: أيام)
- تمكين 2FA إلزامي للمشرفين
- تصدير السجلات تلقائياً

## RTL Support

التطبيق يدعم RTL بالكامل. جميع المكونات مصممة مع مراعاة:
- اتجاه النص
- الـ Padding/Margin
- الـ Icons (قلب الأيقونات في RTL)
- Flex Direction
- الـ Animations (من اليمين لليسار في RTL)
- الـ Sidebar (على اليمين في RTL)
