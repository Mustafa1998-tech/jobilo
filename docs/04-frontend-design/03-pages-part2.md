## Loading, Error & Empty State Components

### Loading Skeleton Pattern
```tsx
// Each page has a matching skeleton component
<ProjectCardSkeleton>  // Pulse animation, same dimensions as card
  <div className="h-4 w-3/4 bg-gray-200 rounded" />
  <div className="h-3 w-1/2 bg-gray-200 rounded mt-2" />
  <div className="h-10 w-full bg-gray-200 rounded mt-4" />
</ProjectCardSkeleton>
```

### Error State Pattern
```tsx
<ErrorState
  icon="⚠️"
  title="حدث خطأ"
  description="لم نتمكن من تحميل البيانات. حاول مرة أخرى."
  action={{
    label: "إعادة المحاولة",
    onClick: retry
  }}
/>
```

### Empty State Pattern
```tsx
<EmptyState
  icon="📋"
  title="لا توجد مشاريع بعد"
  description="ابدأ بإنشاء أول مشروع لك"
  action={{
    label: "نشر مشروع",
    href: "/projects/new"
  }}
/>
```

## Responsive Design Rules

| Element | Desktop (≥1024px) | Tablet (640-1023px) | Mobile (<640px) |
|---------|------------------|--------------------|-----------------|
| Sidebar | Visible, fixed | Collapsible (hamburger) | Bottom nav |
| Project list | 2-3 columns | 2 columns | 1 column |
| Chat | Split view | Split view | Full screen |
| Filters | Side panel | Top bar (collapsible) | Bottom sheet |
| Cards | Full details | Compact | Minimal |
| Tables | Full table | Horizontal scroll | Card list |
| Modals | Center, 600px | Center, 90% | Bottom sheet, full |
| Pagination | Full numbers | Numbers + prev/next | Prev/next only |
| Search bar | Large, visible | Icon + expand | Icon + expand |
| Top bar | Full nav | Icons only | Icons only |
