### Action Buttons
| Button | Visible To | Action |
|--------|-----------|--------|
| تقديم عرض | Freelancer | Scrolls to/opens proposal form |
| حفظ ♡ | Any user | Toggle bookmark |
| الإبلاغ | Any user | Opens report dialog |
| تعديل | Client (owner) | Opens edit page |
| حذف | Client (owner) | Confirmation dialog |
| عرض العروض | Client (owner) | Scrolls to proposals section |
| إغلاق المشروع | Client (owner) | Confirmation modal |
